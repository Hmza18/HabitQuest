HabitQuest is a gamified habit tracker that turns your daily routines into RPG-style quests. Level up your character, maintain streaks, and complete quests to build better habits over time.
Features

- Gamified HUD: See your level, XP, and progress at a glance.
- Habits as Quests: Create habits that feel like quests instead of chores.
- Streak Tracking: Visual weekly view and streak-based feedback.
- Onboarding Wizard: Simple guided setup to define your goals and focus areas.
- Quest Log Panel: Side panel showing today’s quests, streaks, and milestones.
- Toasts & Rewards: XP/badge toasts with undo support for quick corrections.
- Local Persistence: All data is stored locally in your browser.
- Dark/Light Theme: Quick toggle in the top bar.
---
Tech Stack:
- Frontend: React 18 + Vite
- Styling: Custom CSS
- State & Storage: React hooks + `localStorage`
- Effects: `canvas-confetti` for celebrations
